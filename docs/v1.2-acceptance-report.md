# revüe v1.2 Acceptance Report

## Guarantees

1. **Bounded remediation stays practical.** Existing product changes do not trigger the net-new
   creative-production pipeline.
2. **The user is not asked twice.** Explicit premium, flagship, or custom language sets the tier.
3. **Routing is portable and honest.** Capability tiers are vendor-neutral, and unsupported routing is
   omitted rather than fabricated.
4. **Product UI reviews answer the operator's real questions.** The checklist covers current account,
   async feedback, plain score meaning, target, prioritized action, and empty-state recovery.
5. **Codex installation has first-class metadata.** The UI identity and default prompt are shipped in
   `agents/openai.yaml`.
6. **Reported evidence stays honest and actionable.** User reports are not presented as direct
   observations; no-artifact reviews name the missing proof; P0/P1/P2 labels establish fix order.
